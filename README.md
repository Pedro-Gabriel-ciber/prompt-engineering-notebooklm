# 📘 Caderno Temático: Cibersegurança com Engenharia de Prompts (NotebookLM)

---

## 🎯 Contexto e Objetivos

Este projeto foi desenvolvido com foco no estudo de **Cibersegurança**, utilizando técnicas de **Engenharia de Prompts** com o apoio do NotebookLM.

A proposta é utilizar Inteligência Artificial como ferramenta de aprendizagem ativa, permitindo organizar conteúdos, gerar insights e simular cenários reais de segurança da informação.

### Objetivos principais:

- Compreender os fundamentos da cibersegurança;
- Identificar vulnerabilidades e tipos de ataques;
- Utilizar IA para apoiar análise técnica;
- Desenvolver prompts mais eficientes;
- Criar um material reutilizável para estudos futuros.

---

## 🧠 Objetivos de Aprendizado

- Entender conceitos fundamentais de segurança da informação;
- Desenvolver pensamento crítico na análise de respostas da IA;
- Aprender a estruturar perguntas estratégicas (prompts);
- Simular cenários reais de ataques e defesa;
- Criar base para atuação profissional em cibersegurança.

## 📖 Miniguia de Estudo

---

### 🔐 O que é Cibersegurança?

Cibersegurança é o conjunto de práticas e técnicas utilizadas para proteger sistemas, redes e dados contra ataques e acessos não autorizados.

---

### ⚠️ Principais Tipos de Ameaças

- Malware → Software malicioso  
- Phishing → Enganar usuários para roubo de dados  
- Ransomware → Sequestro de informações  
- Força Bruta → Tentativas repetidas de senha  

---

### 🛠️ Principais Vulnerabilidades

- Falta de validação de entrada  
- Senhas fracas  
- Sistemas desatualizados  
- Configuração incorreta  

---

### 🧠 Uso da IA na Cibersegurança

A IA pode ajudar em:

- Simulação de ataques  
- Geração de cenários de teste  
- Análise de vulnerabilidades  
- Identificação de padrões suspeitos  

**Atenção:** Sempre validar as respostas.

---

## 📚 Glossário

- Vulnerabilidade → Falha no sistema  
- Exploit → Forma de explorar uma falha  
- Firewall → Proteção de rede  
- Engenharia Social → Manipulação de pessoas  
- Pentest → Teste de invasão controlado
